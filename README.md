## _Firs Use_

DotNetCoreBackAnd requires [.NET Core](https://dotnet.microsoft.com/download/dotnet/3.1) 3.1 to run.

Install the dependencies.

```sh
write this terminal -> dotnet restore
are you ready,start editing
```
